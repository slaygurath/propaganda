# Release v2.0.0: Circular Letter on the Townshend Acts

This release contains the final version of the circular letter drafted by the Sons of Liberty, arguing that the Townshend Acts are unconstitutional and calling for unified colonial resistance.

## Purpose
The circular letter is intended to be distributed to all colonies, urging them to adopt non‑importation agreements and stand together against parliamentary overreach.

## Changes from previous version
- Strengthened language: replaced "unfortunate imposition" with "blatant usurpation of our rights"
- Added explicit call to action for immediate non‑importation agreements

## Distribution
This document is now ready for printing and dispatch by couriers to the other colonies.

*Released on March 5, 1770*
